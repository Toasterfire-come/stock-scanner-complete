## PayPal Setup Summary

### Step 1: Create PayPal Developer App
1. Go to [PayPal Developer Portal](https://developer.paypal.com/)
2. Create new app: "Trade Scan Pro"
3. Get Client ID and Secret

### Step 2: Create Subscription Plans
Create 3 plans in PayPal dashboard:
- **Bronze:** $9.99/month (1,500 API calls)
- **Silver:** $29.99/month (5,000 API calls)
- **Gold:** $79.99/month (Unlimited API calls)

### Step 3: Configure Webhook
- **URL:** `https://tradescanpro.com/api/billing/webhook/`
- **Events:**
  - BILLING.SUBSCRIPTION.ACTIVATED
  - BILLING.SUBSCRIPTION.CANCELLED
  - BILLING.SUBSCRIPTION.EXPIRED
  - PAYMENT.SALE.COMPLETED
  - PAYMENT.SALE.REFUNDED
