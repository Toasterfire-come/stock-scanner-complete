PHASE 6: STRATEGY RANKING SYSTEM
===============================================================================

6.1 COMPOSITE SCORE ALGORITHM
------------------------------
Score Components (0-100 scale):
  - Performance (30%)
  - Risk Management (25%)
  - Consistency (20%)
  - Trade Efficiency (15%)
  - Community Validation (10%)

Performance Scoring:
  - Annualized Return: 0-10 pts
  - Total Return vs Benchmark: 0-10 pts
  - Profit Factor: 0-10 pts

Risk Management Scoring:
  - Sharpe Ratio: 0-10 pts
  - Max Drawdown: 0-10 pts
  - Recovery Factor: 0-5 pts

Consistency Scoring:
  - Win Rate: 0-8 pts
  - Consecutive Wins/Losses Ratio: 0-7 pts
  - Monthly Return Stability: 0-5 pts

Trade Efficiency Scoring:
  - Avg Win/Avg Loss Ratio: 0-7 pts
  - Trade Frequency: 0-5 pts
  - Commission Impact: 0-3 pts

Community Validation:
  - Clone Count: 0-4 pts
  - User Ratings: 0-4 pts
  - Discussion Activity: 0-2 pts


6.2 LEADERBOARD SYSTEM
-----------------------
Categories:
  - All Strategies
  - Day Trading
  - Swing Trading
  - Long-Term

Timeframes:
  - 1 Month
  - 3 Months
  - 6 Months
  - 1 Year
  - All Time

Filters:
  - Minimum Score
  - Minimum Trades
  - Verified Only
  - My Clones

Sort Options:
  - Composite Score
  - Returns
  - Risk-Adjusted Returns
  - Popularity


6.3 CLONE STRATEGY FEATURE
---------------------------
User Flow:
  1. Click "Clone Strategy" button
  2. Modal presents customization options:
     - Keep/Change symbols
     - Keep/Adjust timeframe
     - Keep/Tune parameters
     - Rename strategy
  3. Clone saved to "My Strategies"
  4. Original strategy gets +1 clone count
  5. Attribution link maintained

Tracking:
  - Original backtest reference
  - Modifications made
  - Clone performance vs original

PHASE 7: EDUCATIONAL PLATFORM
===============================================================================

7.1 COURSE STRUCTURE
--------------------
5 Main Categories:
  1. Trading Fundamentals (6 lessons, ~30 min)
  2. Technical Analysis (15 lessons, ~90 min)
  3. Fundamental Analysis (12 lessons, ~75 min)
  4. Strategy Development (10 lessons, ~60 min)
  5. Psychology & Risk (8 lessons, ~45 min)

Total: 51 lessons, ~5 hours of content


7.2 LESSON COMPONENTS
---------------------
Each Lesson Includes:
  - Title and description
  - Video content (5-15 min)
  - Written content with examples
  - Interactive chart demonstrations
  - Key takeaways summary
  - 3-5 quiz questions
  - Next lesson button

Content Types:
  - Video tutorials
  - Text with diagrams
  - Interactive charts
  - Code examples (for strategies)


7.3 INTERACTIVE TOOLTIPS
-------------------------
Implementation:
  - Hover-triggered popups
  - 200+ trading terms
  - Contextual definitions
  - Related terms links
  - Examples included

Categories:
  - Technical indicators
  - Chart patterns
  - Financial metrics
  - Order types
  - Trading terminology


7.4 TRADING GLOSSARY
---------------------
Features:

  - Usage examples
